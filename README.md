# aoClib

![AoC Logo](https://adventofcode.com/favicon.png)


```bash
make build
./bin/aoc_solve <day>
```
### Project Overview

- `src/main.c`: central runner
- `src/days/`: solutions per day
- `input/`: place input files here (e.g., day4.txt)

Extend by adding new day files in src/days/ and updating main.c.
